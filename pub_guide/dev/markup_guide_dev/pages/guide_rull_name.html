종류 구분 번호 예시

파일 및 폴더

  • 스네이크 표기법(underbar_nate)을 사용한다.

ID Selector

  • 최상위 레이아웃, 폼 요소, 책갈피를 제외하고 id를 사용하지 않는다.
구분 예약어 설명
사이트 레이아웃 .wrapper 사이트 전체영역
.skipNav 바로가기 영역
.header 머리글 영역
.container 페이지 영역
.aside 사이드 영역
.content 본문 영역
.footer 바닥글 영역
#popXXX 레이어팝업 페이지별 ID

Class Selector

  • 체인 선택자의 개수는 3개까지 권장한다.
  • 클래스 예약어는 상황에 맞게 선택적으로 참고한다.
  • 레이아웃에 한하여 정의된 예시이며, 자세한 내용은 설계가이드를 참고한다.
